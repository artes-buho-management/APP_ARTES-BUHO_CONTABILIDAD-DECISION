# SISTEMA CONTABILIDAD IA - RESUMEN VISUAL

## 1) QUE YA ESTA HECHO

- APP VISUAL EN APPS SCRIPT (ROJO + AMARILLO + BLANCO)
- FORMULARIO DE ENTRADA DE DATOS
- PANEL AUTOMATICO CON KPI + GRAFICOS
- ESCENARIOS: OPTIMISTA / BASE / PESIMISTA
- AUDITORIA RAPIDA
- AUDITORIA PROFUNDA DE 1 HORA (POR BLOQUES)
- RESUMEN IA CON GEMINI (SI HAY API KEY)

## 2) DONDE INTRODUCES DATOS

- HOJA: `EntradaDatos`
- CAMPOS: SOLO BLOQUE AMARILLO
- ACCION: MENU -> "Guardar dato desde formulario"

## 3) DONDE VES LA SITUACION

- HOJA: `Panel`
- VES:
  - INGRESOS
  - GASTOS
  - RESULTADO
  - PENDIENTES
  - GRAFICOS POR MES
  - GRAFICOS POR LINEA DE NEGOCIO
  - ESTADO DE AUDITORIA 1H
  - RESUMEN IA

## 4) AUDITORIA ULTRA PROFUNDA

- INICIO: MENU -> "Iniciar auditoria profunda (1 hora)"
- DURACION: MINIMO 60 MIN
- METODO: 12 BLOQUES x 5 MIN
- ESTADO: MENU -> "Estado auditoria profunda"
- SALIDA: HOJA `Auditoria_1h`
